🔧 COMPONENTES PRINCIPAIS
═══════════════════════════════════════════════════════════

1. OLLAMA CLIENT (src/ollama_client.py)
   ├─ Conexão com Ollama local
   ├─ Análise de relevância com IA
   ├─ Geração de embeddings
   ├─ Resumos automáticos
   └─ Análise cruzada entre grupos

2. TELEGRAM MONITOR (src/telegram_monitor.py)
   ├─ Conexão com API do Telegram
   ├─ Monitoramento em tempo real
   ├─ Coleta de histórico
   ├─ Filtro inteligente de mensagens
   └─ Handler de eventos

3. DATABASE (src/database.py)
   ├─ Gerenciamento SQLite
   ├─ Armazenamento de mensagens
   ├─ Queries otimizadas
   ├─ Cleanup automático
   └─ Estatísticas

4. REPORT GENERATOR (src/report_generator.py)
   ├─ Relatórios diários
   ├─ Análise por keyword
   ├─ Perfis de usuários
   ├─ Análise cruzada
   └─ Exportação Markdown
